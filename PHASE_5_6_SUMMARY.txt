🎯 Phase 5: Metrics Functions (8개)
─────────────────────────────────────────────────────────────────────────────

분류 메트릭:
  1. sklearn_accuracy_score(y_true, y_pred) → number
     - 정확하게 예측한 비율: correct / total
     - 범위: 0.0 ~ 1.0

  2. sklearn_confusion_matrix(y_true, y_pred) → [[tn, fp], [fn, tp]]
     - 이진 분류 성능 행렬
     - [[True Negatives, False Positives], [False Negatives, True Positives]]

  3. sklearn_precision(y_true, y_pred) → number
     - 양성 예측의 정확도: TP / (TP + FP)
     - 거짓 양성 최소화 시 사용

  4. sklearn_recall(y_true, y_pred) → number
     - 양성 케이스 감지율: TP / (TP + FN)
     - 거짓 음성 최소화 시 사용

  5. sklearn_f1_score(y_true, y_pred) → number
     - Precision과 Recall의 조화 평균
     - 불균형 데이터셋에 유용

회귀 메트릭:
  6. sklearn_mse(y_true, y_pred) → number
     - 평균 제곱 오차: Σ(y_true - y_pred)² / n
     - 큰 오차에 큰 페널티

  7. sklearn_mae(y_true, y_pred) → number
     - 평균 절대 오차: Σ|y_true - y_pred| / n
     - 이상치에 덜 민감

  8. sklearn_r2_score(y_true, y_pred) → number
     - 결정 계수: 1 - (SS_res / SS_tot)
     - 범위: -∞ ~ 1.0 (1.0=완벽, 0.0=평균 수준)

🔧 Phase 5-A: Preprocessing (4개)
─────────────────────────────────────────────────────────────────────────────

  1. sklearn_standard_scaler_fit(X) → { means, stds, n_features }
     - 정규화 파라미터 학습
     - 각 특성의 평균과 표준편차 계산

  2. sklearn_standard_scaler_transform(X, params) → X_scaled
     - 학습된 파라미터로 정규화
     - 공식: (x - mean) / std

  3. sklearn_standard_scaler_fit_transform(X) → X_scaled
     - fit과 transform을 한 번에 수행

  4. sklearn_train_test_split(X, y, test_size) → split object
     - 데이터를 학습/테스트 세트로 분할
     - Fisher-Yates 셔플 알고리즘 사용

🤖 Phase 6-A: Linear Models (4개)
─────────────────────────────────────────────────────────────────────────────

  1. sklearn_linear_fit(X, y, lr=0.01, epochs=100) → model
     - 선형 회귀 학습 (경사 하강법)
     - 반환: { weights, bias, loss_history }

  2. sklearn_linear_predict(X, model) → predictions
     - 선형 회귀 예측
     - 공식: y = w·x + b

  3. sklearn_logistic_fit(X, y, lr=0.01, epochs=100) → model
     - 로지스틱 회귀 학습 (이진 분류)
     - 시그모이드 활성화 + 이진 크로스엔트로피 손실

  4. sklearn_logistic_predict(X, model) → predictions
     - 로지스틱 회귀 예측 (확률)
     - 반환: 0.0~1.0 범위의 확률

🎯 Phase 6-B: Clustering (2개)
─────────────────────────────────────────────────────────────────────────────

  1. sklearn_kmeans_fit(X, k=3, max_iter=100, random_state=42) → model
     - K-Means 클러스터링
     - Lloyd 알고리즘으로 중심점 반복 업데이트
     - 반환: { centers, labels, inertia }

  2. sklearn_kmeans_predict(X, model) → labels
     - 학습된 K-Means로 새 데이터 분류

👥 Phase 6-C: Neighbors (2개)
─────────────────────────────────────────────────────────────────────────────

  1. sklearn_knn_fit(X, y) → model
     - KNN 모델 학습 (게으른 학습)
     - 학습 데이터를 메모리에 저장

  2. sklearn_knn_predict(X, model, k=3) → predictions
     - KNN 예측 (k개 가장 가까운 이웃으로 분류)
     - 유클리드 거리 기반, 다중 클래스 지원

📊 성능 분석
─────────────────────────────────────────────────────────────────────────────

시간 복잡도:
  메트릭 함수:              O(n) - 단일 통과
  Preprocessing:            O(n×d) - n샘플, d특성
  Linear fit:              O(epochs × n × d)
  KNN predict:             O(m × n × d) - m테스트, n훈련 샘플
  K-Means:                 O(iter × k × n × d)

공간 복잡도:
  메트릭:                  O(1)
  Scaler:                  O(d)
  모델 저장:               O(d) ~ O(n×d)

실제 성능:
  1000개 샘플:             <1초
  10000개 샘플:            ~5초
  100000개 샘플:           ~50초

  let predictions = linear_predict(split["X_test"], model);
  let r2 = r2_score(split["y_test"], predictions);

  let cm = confusion_matrix(y, probs > 0.5 ? 1 : 0);
