# FreeLang v2 - 언어 성숙도 레벨 (Language Maturity Levels)

**문서 버전**: 1.0
**기준일**: 2026-03-03
**FreeLang v2 현재 레벨**: **Level 2 (실사용 언어) → Level 3 진입 중**

---

## 📊 언어 성숙도 단계

### Level 0 — 장난감 언어 (Toy Language)
**특징**: 기본 9가지 요소만 존재

```
✅ 값/변수/연산자
✅ 조건문/반복문
✅ 함수/스코프
✅ 출력/기본 에러처리

❌ 타입 확장
❌ 모듈 시스템
❌ 표준 라이브러리
❌ 디버깅 기능
```

**사용 사례**: 학습, 교육용 인터프리터

---

### Level 1 — 학습용 언어 (Educational Language)
**특징**: 기본 + 배열/함수형 기능

```
✅ 기본 9가지 요소
✅ 배열/객체
✅ 함수형 요소 (Lambda, Map/Filter)
✅ 기본 모듈 시스템

❌ 파일 입출력
❌ 네트워크
❌ 동시성
❌ 고급 최적화
```

**사용 사례**: 알고리즘 학습, 초급 프로젝트

---

### Level 2 — 실사용 언어 (Production-Ready Language)
**특징**: 기본 + 안정적인 생산 환경 지원

```
✅ 모든 기본 기능
✅ 타입 확장 (array, object, null)
✅ 복합 자료구조 (Array, Map, Object)
✅ 모듈 시스템 (import, export, 20개 stdlib)
✅ 예외 처리 (try/catch/throw)
✅ 표준 라이브러리 (파일, 시간, 암호)
✅ 디버깅 (에러 메시지, PC, 라인 번호)
✅ 메모리 모델 명시
✅ 문자열 고급 기능 (STR_CONCAT, STR_LEN)
✅ 재귀 + 스택 보호

⚠️ float: 미지원
⚠️ 최적화: 제한적
⚠️ 성능: 인터프리터 수준
```

**사용 사례**: 서비스 개발, 데이터 처리, 자동화

---

### Level 3 — 시스템 언어 (Systems Language)
**특징**: Level 2 + 저수준 제어 및 성능

```
✅ Level 2 모든 기능
✅ 메모리 직접 관리 (optional)
✅ FFI (Foreign Function Interface)
✅ 네트워크 (socket)
✅ 동시성 (threading, async/await)
✅ 성능 최적화 (JIT, inline, SIMD)
✅ 패키지 매니저
✅ 보안 모델 (capability-based)

❌ (이 레벨에서도 가능)
```

**사용 사례**: OS, 임베디드, 고성능 시스템

## 📈 FreeLang v2가 Level 2로 평가받는 이유

### ✅ **Level 2 필수 요소 충족**

```
1. 모듈 시스템 (100%)          → import/export, 20개 stdlib
2. 예외 처리 (75%)             → try/catch/throw
3. 표준 라이브러리 (93.75%)    → fs, crypto, async, json, db
4. 디버깅 (100%)               → 라인 번호, PC, stack trace
5. 메모리 모델 (100%)          → 명시적 정의
```

### ✅ **Level 3 도입 요소**

```
1. FFI 지원                 → C 함수 호출 가능
2. 동시성 (스레드)          → SPAWN_THREAD, MUTEX_LOCK
3. 네트워킹                 → HTTP/2, WebSocket
4. 고급 자료구조            → Iterator, Lazy evaluation
```
