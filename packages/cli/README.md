# @freelang/cli

FreeLang 커맨드라인 인터페이스

## 설치

```bash
npm install @freelang/cli
```

## 사용법

```bash
freelang [명령어] [옵션]
```

### 주요 명령어

- `freelang build` - 코드 컴파일
- `freelang run` - 코드 실행
- `freelang test` - 테스트 실행
- `freelang format` - 코드 포매팅
- `freelang lint` - 린트 검사

## 예제

```bash
# 파일 컴파일
freelang build main.fl

# 파일 실행
freelang run main.fl

# 테스트 실행
freelang test
```
